garage-io
=========

Garage control and monitoring system. Uses Arduino > serial > python.  

Currently sends push notification via Prowl for iOS and plays alarm sound via Raspberry PI headphone jack.

## Prerequisites:

pySerial  
redis-py  
bottle.py  

## Optional:
PyNMA
